IMPORT LINK TEXT FILES [1]
PATHLOSS RADIO AND ANTENNA FILES CROSS REFERENCE [2]
A radio and antenna data files cross reference has been added to the Import link text files operation. This feature allows an interference calculation to be carried out immediately following the import. External text files are used to cross reference the radio and antenna identifiers used in the import data to the Pathloss radio and antenna code files. [3]
The Pathloss code files are used to fill in any missing information in the import data. For example if the import data contains only the transmit power and the radio identifier, the receiver threshold information would be read from the cross referenced Pathloss radio code file, but the transmit power would not be changed. [4]
The cross reference files are ASCII text files which meet the following requirements: [5]
• The file names must be “ant_xref.txt” and “rad_xref.txt” for the antennas and radios cross reference files respec- [6]
tively. These files must be located in the same directory that the Pathloss program is installed in. [7]
• Each line in the file must consist of the import data descriptor in double quotes followed by the Pathloss code file [8]
name separated by a space or tab. [9]
The comparison in the cross reference is not case sensitive; however the spelling in the cross reference file must [10]
be the same as the data file. If there are several variants in the import data file, then a cross reference line must [11]
be created for each variant. The same Pathloss code file name could be used for each variant. [12]
An example of a radio and antenna cross reference file is given below. [13]
ant_xref.txt [14]
"15_GHz/0.6m Andrew" 6962 [15]
"15_GHz/1.2m Andrew" 6964 [16]
"15_GHz/0.3m Andrew" 6961 [17]
"18_GHz/0.6m ANDREW" 6967 [18]
"18_GHz/0.3m ANDREW" 7010 [19]
"18_Ghz/1.2m ANDREW" 4504a [20]
rad_xref.txt [21]
"Ajax radio AJ15, 8x2 guaranteed" AJ15_16s_8E1 [22]
"Ajax radio AJ15, 4x2 guaranteed" AJ15_4s_4E1 [23]
"Ajax radio AJ15, 16x2 guaranteed" AJ15_16s_16E1 [24]
"Ajax radio AJ18, 8x2 guaranteed" AJ18_16s_8E1 [25]
"Ajax radio AJ18, 4x2 guaranteed" AJ18_4s_4E1 [26]
"Ajax radio AJ18, 16x2 guaranteed" AJ18_16s_16E1 [27]
The radio and antenna data files must be available for the conversion to succeed. The location of the files must be identified. Click Configure - Directories - Microwave Antenna Codes / Microwave Radio Codes and point to the top most directory which contains the data files [28]
