Pathloss Version 4.0 File Association [1]
The pathloss installation program did not complete the Windows registry entry correctly on shipments before August 1999. Double clicking on a pathloss file (pl4) would start the program but did not load the file. The procedure to edit the registry and correct this problem is given in the steps below. [2]
Click Start - Run and enter the command line REGEDIT. Click OK to start the windows [3]
registry editor program. [4]
Select Edit - Find.. on the [5]
menu bar and enter the [6]
search string pl4file. [7]
The correct location is shown in the directory structure on the right. [8]
Double click on the “ab” icon [9]
and edit the entry as shown. [10]
The %1 is the missing [11]
portion. The path name will [12]
be the directory that the [13]
pathloss program was [14]
installed in. [15]
Click OK and close [16]
REGEDIT program. [17]
